## 参与方式

- Star / Watch 项目，关注版本演进和治理模板更新。
- 通过 GitHub Issues 提交工作流、门禁、脚手架、文档或跨 Agent 协作问题。
- 通过 PR 贡献治理模板、验证 profile、资源治理规则、示例项目或真实落地案例。
- 国内用户可通过 Gitee 镜像同步查看和反馈。

# skill: task-breakdown

## Phase
PLAN

## Purpose
Break approved plan into atomic, executable tasks.

## Triggers
- Keywords: task, breakdown, decompose
- After: plan approved

## Procedure

1. **Read Plan Modules**
   - Extract module list from plan
   - Identify dependencies between modules

2. **Create Tasks**
   - Each module → 1-3 tasks
   - Define filesInvolved per task
   - Set dependsOn for dependency order

3. **Estimate Effort**
   - Token budget per task
   - Duration estimate

4. **Assign Roles**
   - Match task to agent profile
   - Set requiredCapabilities

## Output
- Task artifacts linked to Plan
- Ready for BUILD phase
